# Selamat Datang di Wiki Sinergi PAS 🏢

**Sinergi PAS** (*Sistem Informasi Kinerja Pemasyarakatan*) adalah platform *Enterprise Resource Planning* (ERP) internal yang dikembangkan untuk **Lapas Jombang**. Sistem ini dirancang untuk mendigitalisasi manajemen dokumen, pelaporan masalah, dan pemantauan kinerja pegawai secara efisien dan transparan.

## 📖 Navigasi Wiki
Silakan gunakan menu di bawah ini untuk mempelajari lebih lanjut tentang sistem:

1.  **[Panduan Instalasi](Instalasi-&-Setup)**: Langkah-langkah teknis untuk menjalankan proyek di server lokal atau produksi.
2.  **[Panduan Administrator](Panduan-Admin)**: Instruksi lengkap bagi Superadmin untuk mengelola data pegawai dan verifikasi dokumen.
3.  **[Panduan Pegawai](Panduan-Pegawai)**: Instruksi bagi pegawai untuk mengelola profil, mengunggah dokumen, dan melaporkan kendala.
4.  **[Arsitektur & Database](Arsitektur-&-Database)**: Penjelasan mengenai skema database, teknologi yang digunakan, dan alur logika sistem.
5.  **[Troubleshooting](Troubleshooting)**: Solusi untuk masalah umum yang mungkin muncul saat instalasi atau penggunaan.
